Setelah kemarin kita telah mengenal macam-macam tag yang ada pada html,
ditutorial kali ini kita akan membuat web static dengan menggunakan html.

Penasaran, silahkan simak tutorial dibawah ini:

  1. Buka aplikasi termux anda dan ketikkan perintah dibawah ini: 
    2.     $ lampp start
  2. Buat sebuah direktori baru di folder /sdcard/www menggunakan perintah dibawah ini: 
    3.     $ cd /sdcard/www
    $ mkdir -p webstatic
  3. Buka text editor dan buat file baru didirektori webstatic yang tadi telah kita buat, dengan nama index.html dan ketikkan perintah dibawah ini: 
    4.     <!DOCTYPE html>
