Setelah kemarin kita telah belajar html untuk pemula, sekarang kita akan melanjutkan tutorial berikutnya yaitu belajar tag-tag yang ada pada html.
Tag html sendiri sangat banyak dan akan menghabiskan banyak waktu kalau dibahas satu-persatu.

Maka dari itu saya akan memberikan sebuah referensi saja mengenai tag-tag html beserta fungsinya.

Referensi tag HTML dibawah ini juga termasuk tag HTML5 yang terakhir di release oleh W3C. Dibawah akan disebutkan kegunaan masing-masing tag juga penggunaan tag yang harus di hindari karena sudah tidak lagi disupport HTML5.

Berikut adalah tag-tag html:
Nama Keterangan atau Fungsi
<!DOCTYPE html> Tag untuk menentukan tipe dokumen.
<html></html> Tag untuk membuat sebuah dokumen HTML
<head></head> Tag yang berisi bagian untuk menentukan judul website, meta dari website, stylesheet website.
<title></title> Tag untuk membuat judul website
