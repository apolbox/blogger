caranya agar dalam post atau update artikel kita lebih mudah.
</p>
<b>Syarat:</b>
<ol>
    <li>Mempunyai pengetahuan tentang HTML</li>
    <li>Aplikasi Termux (playstore)</li>
    <li>Akun <a href="https://github.com">github</a></li>
    <li>Text editor / notepad</li>
</ol>
<p>
Bagi yang belum tahu tentang HTML lebih baik belajar terlebih dahulu,
agar dalam mengikuti tutorial ini lebih mudah dipahami.<br/>
Oke tanpa banyak basa-basi mari kita mulai.
</p>
<ol>
    <li>Pastikan anda telah mempunyai akun github,<br/>
    Jika sudah buat repository baru beri nama <code>blogger</code>,<br/>
    lalu buat sebuah file dengan nama <code>post-1.txt</code> (anda bebas memberi nama filenya)
    lalu isi file <code>post-1.txt</code> dengan tulisan anda.<br/>
    </li>
    <li>
    Buka aplikasi termux ketikan perintah berikut:<br/>
    <pre style="width=auto; background: #f2f2f2; padding: 12px; border-radius: 2px;">
    $ apt update && apt upgrade
    $ apt install git
    $ termux-setup-storage
    $ cd /sdcard
    $ git clone https://github.com/nama_akun_github/repository_anda
    </pre>
    </li>
    <li>
    Buka notepad atau text editor kesayangan anda, lalu buka file <code>post-1.txt</code>
    yang tadi telah kita buat dan isi file dengan tulisan anda.

    contoh isi file <code>post-1.txt</code>:
    <pre style="width=auto; background: #f2f2f2; padding: 12px; border-radius: 2px;">
    Ini artikel pertamaku
    </pre>
    </li>
    <li>Simpan dan buka kembali aplikasi termux<br/>
    Setelah termux terbuka ketikkan perintah berikut:
    <pre style="width=auto; background: #f2f2f2; padding: 12px; border-radius: 2px;">
    $ cd /sdcard
    $ cd nama_repository_anda
    $ git add *
    $ git commit -m "Artikel pertamaku"
    $ git push
    </pre>
    Jika muncul pertanyaan untuk memasukkan username dan password, isi dengan username dan password akun github anda.
    </li>
    <li>
    Buka blogger anda dibrowser kesayangan anda, lalu buat entri baru.
    Anda tinggal copy dan paste saja kode dibawah ini:
    <pre style="width=auto; background: #f2f2f2; padding: 12px; border-radius: 2px;">
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>
    <script src="https://unpkg.com/axios/dist/axios.min.js"></script>
    <script>
        // Isi url artikel anda
        const Url='https://raw.githubusercontent.com/apolbox/blogger/master/termux/webserver.txt';

        // Jquery run
        $(document).ready(function(){
            // Beri salam hangat pada pembaca
            $("#pages-header").text("Selamat gini hari agan semua");
                axios.get(Url, { 'responseType': 'text/plain'})
                .then(response => {
                    $("#pages-body").html(response.data);
                })
                .catch(err=>console.log(err))
        });
    </script>
    </pre>
    </li>
</ol>
