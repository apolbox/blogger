Oke kita anggap termux sudah berhasil kalian install.
mari kita mulai langkah-langkahnya.

1. Lakukan update && upgrade
$ apt update && apt upgrade

2. Install git
$ apt install -y git

3. Cloning repository dari github
$ git clone https://github.com/apolbox/termux-webserver.git
$ cd termux-webserver
$ chmod +x ./install
$ ./install

Proses instalasi akan berjalan, tunggu sampai selesai. Jika muncul pertanyaan anda bisa memilih untuk mengaturnya atau tidak.

4. Untuk menginstall database gunakan perintah berikut:
$ lampp-db install

Proses instalasi akan berjalan dan akan muncul pertanyaan untuk menjalankan layanan database.

Untuk menjalankan layanan webserver dan layanan database anda bisa menggunakan perintah berikut:
$ lampp start && lampp-db start

Untuk menghentikan semua layanan gunakan perintah berikut:
$ lampp stop && lampp-db stop

Sekarang buka web browser anda dan coba ketikkan localhost:8080
